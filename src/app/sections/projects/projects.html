AFARS Mobile Application

UI/UX design

Flight Attendants use this app to create report/issue while checking the flight. FA checks the flight details in sequence tab and creates report. AFARS App consumes Flight Hub Web Services to retrieve sequences. It uses SQLite local Database to store sequences. Report/issues and store it on the local DB. On getting internet connection Flight attendant uses this app to transmit the reports to the server. This app is developed using HTML5 + Angular JS + bootstrap + Cordova (+ Grunt + Bower) for Android and iOS platform. -Developed while working at Hewlett Packard Enterprise
